# Material UI Responsive Components

Three fully responsive UI implementations using Material UI:

## Components

### 1. Landing Page
- Responsive Grid layout
- Sections stack on mobile, side-by-side on desktop
- Breakpoint-based column adjustments

### 2. Dashboard
- Responsive navbar and collapsible sidebar
- Card grid with automatic column adjustment
- Mobile-first responsive design

### 3. Admin Panel
- Light/dark theme switching
- Styled component overrides
- Multi-panel layout that collapses on mobile

## Setup

```bash
npm install
npm run dev
```

## Features
- Material UI Grid system
- Styled-components integration
- Theme Provider with custom overrides
- Responsive breakpoints (xs, sm, md, lg)
